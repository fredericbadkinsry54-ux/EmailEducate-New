# **App Name**: EmailEducate

## Core Features:

- Strategy Lessons: Deliver bite-sized email marketing lessons to users.
- Tool Recommendations: Suggest suitable email marketing tools (e.g., Mailchimp, ConvertKit) based on the lesson content.  A tool should use AI to decide when it is appropriate to present tool recommendations based on the current lesson.
- Progress Tracking: Allow users to track their lesson progress.
- Signup Form: Allow users to sign up for the EmailEducate app

## Style Guidelines:

- Primary color: Indigo (#4F46E5) to align with the source website's branding and convey a sense of professionalism.
- Background color: Very light lavender (#F5F3FF), a desaturated variant of the primary hue, to create a clean and unobtrusive backdrop.
- Accent color: Violet (#7C3AED), an analogous hue, to provide visual interest and highlight important elements.
- Body and headline font: 'Inter', a sans-serif font with a modern and neutral look, suitable for both headlines and body text.
- Use clean, professional icons to represent email marketing concepts.
- Maintain a clean, modern layout with plenty of white space to enhance readability and focus.
- Subtle animations when marking lessons as complete
